# Gas Leakage Detection System using ESP8266

## 📌 Description

This project detects gas leakage using a gas sensor and NodeMCU ESP8266 and provides real-time alerts.

## ⚙️ Components

* NodeMCU ESP8266
* Gas Sensor (MQ series)
* LCD Display
* Buzzer
* LED

## 🚀 Features

* Real-time gas monitoring
* Automatic alert system
* Wi-Fi connectivity
* LCD display output

## 📊 Applications

* Home safety
* Industrial safety
* Smart IoT systems
